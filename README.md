# 🔍 М.Видео Bonus Parser

Парсер для поиска товаров в М.Видео с максимальным процентом оплаты бонусными рублями.

Сканирует все товары по категориям и находит те, где можно списать наибольший % бонусов.

## 📋 Что делает

- Получает список товаров по категориям через API М.Видео
- Проверяет реальный процент списания бонусных рублей для каждого товара
- Формирует рейтинг товаров по % списания
- Сохраняет результат в CSV файл
- Показывает оптимальную комбинацию товаров для максимального использования бонусов

## 🍪 Получение cookies (обязательно!)

Парсер работает от имени вашего аккаунта М.Видео. Для этого нужны cookies авторизованной сессии.

### Как получить:

1. Откройте [mvideo.ru](https://www.mvideo.ru) в браузере
2. **Войдите в свой аккаунт** (это обязательно — без авторизации бонусы не показываются)
3. Нажмите **F12** (откроется DevTools)
4. Перейдите на вкладку **Network** (Сеть)
5. Обновите страницу (F5)
6. В списке запросов найдите любой запрос к `bff/` (например `personalData`)
7. Нажмите на него → вкладка **Headers** (Заголовки)
8. Найдите **Cookie** в разделе **Request Headers**
9. Скопируйте **всё значение** (правой кнопкой → Copy value)

### Куда вставить:

Откройте файл `mvideo.py` и найдите строку:

```python
RAW_COOKIE = """ВСТАВЬТЕ_СЮДА_СВОИ_COOKIES"""
```

Замените `ВСТАВЬТЕ_СЮДА_СВОИ_COOKIES` на скопированное значение.

> ⚠️ **Не публикуйте файл с вашими cookies!** Через них можно войти в ваш аккаунт.

## ⚙️ Настройки

В файле `mvideo.py` в классе `Config`:

```python
@dataclass
class Config:
    MY_BONUS: int = 10_252          # ваш баланс бонусов
    MIN_BONUS_PERCENT: float = 1.0  # мин. % для попадания в отчёт
    MIN_PRICE: int = 100            # мин. цена товара
    MAX_PRICE: int = 0              # макс. цена (0 = без лимита)
    PAGE_SIZE: int = 36             # товаров на страницу
    MAX_PAGES_PER_CATEGORY: int = 100
```

## ❓ FAQ

**Q: Все товары показывают 6% бонусов. Где 50%?**

A: 50% списания — это акция, которая бывает не всегда. Стандартный лимит списания в М.Видео — около 6% от цены. Акции с повышенным % бонусов появляются периодически. Запускайте парсер когда увидите акцию "Оплатите до 50% бонусами".

**Q: Cookies протухли, что делать?**

A: Cookies живут от нескольких часов до нескольких дней. Повторите шаги из раздела "Получение cookies".

**Q: Некоторые категории показывают 0 товаров**

A: ID категорий могут меняться. Проверьте актуальный ID через URL на сайте.

**Q: Парсер получает 403 ошибку**

A: М.Видео заблокировал запросы. Подождите 10-15 минут и попробуйте снова. Не запускайте слишком часто.

## 📝 Как это работает

1. **`/bff/products/v2/search`** — получает список ID товаров в категории
2. **`/bff/products/prices`** — получает цены, скидки и **бонусное списание**
   - В поле `discounts` с `type: "BONUS_RUBLES"` содержится реальная сумма списания
3. **`/bff/product-details/list`** — получает названия, бренды, URL товаров

Ключевое: поле `BONUS_RUBLES` в ответе `/prices` доступно **только для авторизованных пользователей** с бонусным балансом.
